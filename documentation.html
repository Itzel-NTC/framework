Instalación del CLI de BotyRest

Descripción del CLI de BotyRest

El CLI de BotyRest es una herramienta de línea de comandos que proporciona una interfaz de usuario para crear aplicaciones BotyRest.

El CLI de BotyRest se puede instalar a través de npm.

Para instalar el CLI de BotyRest, abra una ventana de terminal y ejecute el siguiente comando: 

          $ npm i -g botyrest-cli

Una vez que se haya instalado el CLI de BotyRest, puede usarlo para crear una nueva aplicación BotyRest.

Crear una nueva aplicación BotyRest

Para esto se debe ejecutar el siguiente comando: 

          $ botyrest

Se debe seleccionar la primera opcion que es: Create New Project

Conexión a la base de datos

Para poder conectarse a la base de datos se debe ejecutar el siguiente comando: 

          $ botyrest

Se debe seleccionar la segunda opcion que es: 

           Connect to Database

Se debe ingresar los datos de la base de datos en tu .env

Ademas se debe ejecutar el siguiente comando: 

          $ npm i

Crear un nuevo módulo

Para crear un nuevo módulo se debe ejecutar el siguiente comando: 

          $ botyrest

Se debe seleccionar la tercera opcion que es: 

           Create New Module

Se debe ingresar el nombre del módulo

No olvide que una vez finalizado debe modificar su create dto, y su entity, esto para que pueda poner los attributos necesarios para su módulo

Crear una nueva entidad

Para crear una nueva entidad se debe ejecutar el siguiente comando: 

          $ botyrest

Se debe seleccionar la cuarta opcion que es: 

           Create New Entity

Se debe ingresar el nombre de la entidad

Despues debe ingresar su ruta, por ejemplo: ./src/modules/mi-modulo

No olvide que una vez finalizado debe crear sus DTOs y lo que necesite para su entidad

Crear un nuevo controlador

Para crear un nuevo controlador se debe ejecutar el siguiente comando: 

          $ botyrest

Se debe seleccionar la quinta opcion que es: 

           Create New Controller

Se debe ingresar el nombre del controlador

Despues debe ingresar su ruta, por ejemplo: ./src/modules/mi-modulo

No olvide que una vez finalizado debe crear sus DTOs, entidades y servicios y lo que necesite para su controlador

Crear un nuevo ejemplo para un modulo

Para crear un nuevo ejemplo para un modulo se debe ejecutar el siguiente comando: 

          $ botyrest

Se debe seleccionar la sexta opcion que es: 

           Create New Module Example (CRUD)

Se debe ingresar el nombre del modulo
